# beforeExport( )

Called before export.

Called before export.

It stores the authentication data, to be used to export the database, in a temporary file.

For security reasons, it's recommended to specify the password in a configuration file and not in the command (a user can execute a ps aux | grep mysqldump and see the password). So it creates a temporary file to store the configuration options
